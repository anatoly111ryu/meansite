Overview

How It works

  1. The Browser connects to the desired url which is in the form of http://localhost:3000 (if you're in a development environment) or the site name.
    The Connection is handled by Express which uses the consolidate.js library to render the server side application markup (the default templating engine is swig).
  2. Page is returned to browser - The page rendered contains the inclusion of the angular.js library(among others). The markup contains an "ng-view" tag which is where the angular app initializes.
